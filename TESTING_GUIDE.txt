 AI AGENTS SUMMARY (for judges/demo)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Agent 1 — Image Classifier   → classifyIssue.js
            Gemini Vision reads photo → category, severity, title, description

  Agent 2 — Duplicate Detector → duplicateDetector.js
            GPS radius + Gemini Vision compares images → duplicate/unique

  Agent 3 — Priority Engine    → priorityEngine.js
            7-factor formula → Critical/High/Medium/Low score

  Agent 4 — Smart Router       → routeIssue.js
            Gemini routes to correct department + escalation logic

  Agent 5 — Resolution Verifier→ verifyIssue.js
            Gemini compares before/after → Resolved/Partial/Not Resolved

  Agent 6 — AI Chat Assistant  → chatAgent.js
            Gemini with DB context → answers civic queries

  Agent 7 — Predictive Hotspots→ predictiveAgent.js + analyticsAgent.js
            DB aggregations → Gemini → risk maps, seasonal forecasts

  Agent 8 — Action Recommender → recommendationAgent.js
            Pattern detection (clusters, day trends) → systemic actions
